# Background Music

Place the following music files in this directory:

## Required Music Tracks (.ogg format recommended)

| Filename | Description | Loop |
|----------|-------------|------|
| `menu.ogg` | Main menu background music | Yes |
| `gameplay.ogg` | In-game background music | Yes |

## Optional Additional Tracks

| Filename | Description | Loop |
|----------|-------------|------|
| `victory.ogg` | Level complete celebration | No |
| `gameover.ogg` | Game over music | No |
| `endless.ogg` | Endless mode music | Yes |

## Audio Specifications

- **Format**: OGG Vorbis (preferred) or MP3
- **Sample Rate**: 44100 Hz
- **Bit Rate**: 128-192 kbps
- **Duration**: 1-3 minutes (for looping tracks)

## Music Style Guidelines

- **Menu**: Calm, ambient, inviting
- **Gameplay**: Upbeat, energetic but not distracting
- **Victory**: Triumphant, celebratory
- **Game Over**: Melancholic but not depressing

## Licensing Notes

Ensure all music files are either:
1. Original compositions
2. Licensed for commercial use (verify mobile game licensing)
3. Royalty-free with appropriate license

**Important**: Keep documentation of all music licenses for App Store/Play Store submissions.
