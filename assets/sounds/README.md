# Sound Effects

Place the following sound effect files in this directory:

## Required Sound Effects (.wav format recommended)

| Filename | Description | Duration |
|----------|-------------|----------|
| `collision.wav` | Object collision sound | 0.1-0.3s |
| `goal.wav` | Object reaching goal zone | 0.3-0.5s |
| `swipe.wav` | Gravity stroke drawing | 0.2-0.4s |
| `chain.wav` | Chain reaction triggered | 0.3-0.5s |
| `complete.wav` | Level completion fanfare | 1.0-2.0s |
| `star.wav` | Star earned sound | 0.2-0.3s |
| `click.wav` | Button click | 0.05-0.1s |
| `select.wav` | Menu selection | 0.1-0.2s |
| `warning.wav` | Time warning beep | 0.2-0.3s |
| `gameover.wav` | Game over sound | 0.5-1.0s |
| `energy.wav` | Energy transfer | 0.2-0.3s |
| `bounce.wav` | Object bouncing off wall | 0.1-0.2s |

## Audio Specifications

- **Format**: WAV (PCM) or OGG
- **Sample Rate**: 44100 Hz
- **Bit Depth**: 16-bit
- **Channels**: Mono or Stereo

## Licensing Notes
